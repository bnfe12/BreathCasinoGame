## 1. СТРУКТУРА ИГРЫ

3 Раунда. Раунд длится до смерти одного игрока. Внутри раунда — бесконечные мини-раунды.
Мини-раунд заканчивается когда барабан пуст → новые патроны → новый мини-раунд.
Так до тех пор пока HP одного из игроков не достигнет 0.

| Параметр | Раунд 1 | Раунд 2 | Раунд 3 |
|---|---|---|---|
| HP старт | 2 | 3 | 4 |
| Патронов / мини-раунд | 3 | 3–5 | 4–6 |
| O2 старт | 60s | 60s | 60s |
| Special карты | НЕТ | ДА | ДА |
| Explosive патрон | НЕТ | НЕТ | 1 шт. опц. |

Механики идентичны во всех раундах. Меняются только цифры и доступные инструменты.

## 2. МИНИ-РАУНД — ФАЗЫ

Строгая последовательность. Фазы не пропускаются и не меняются местами.

```
BulletReveal → Dealing → Attack → Defense → Resolution → Shooting → NextMiniRound
```

| Фаза | Что происходит |
|---|---|
| BulletReveal | Барабан кратко открывается. Игрок видит СОСТАВ (типы и количество). Порядок скрыт. Закрывается и перемешивается. |
| Dealing | Тумба поднимается на шестернях. Раздаются карты + билеты обоим игрокам. Игрок берёт карту с тумбы → +5s O2 за каждую. |
| Attack | Атакующий кладёт карту в AttackSlot → +10s O2. |
| Defense | Защитник кладёт карту в DefenseSlot → +10s O2. Правило: вес защиты >= вес атаки. |
| Resolution | Сравниваются веса. Определяется результат. Выдаются пистолет и билеты. |
| Shooting | Владелец пистолета выбирает цель: враг или себя. |
| NextMiniRound | Барабан пуст → StartMiniRound. HP = 0 → конец Раунда. |

**ЖЁСТКОЕ ПРАВИЛО:** Атака ТОЛЬКО в AttackSlot. Ответ ТОЛЬКО в DefenseSlot.
Атака на атаку физически невозможна. Никаких исключений.

## 3. ПАТРОНЫ

Минимум 1 Blank гарантирован в каждом мини-раунде ВСЕГДА.
После каждого выстрела патрон извлекается из барабана.
Барабан пуст → StartMiniRound.

| Тип | Урон | Доступность |
|---|---|---|
| Blank (Холостой) | 0 | Все раунды. Мин. 1 гарантирован. |
| Live (Боевой) | 1 HP | Все раунды. |
| Explosive (Взрывной) | 2 HP | Только Раунд 3. Максимум 1 шт. опц. |

DoubleDamage + Explosive в Раунде 3 = 4 HP = мгновенная смерть (HP Раунда 3 = 4).

## 4. КАРТЫ

| Тип | Вес | Цвет | Доступность |
|---|---|---|---|
| Resource | 2–8 | Серый | Все раунды |
| Threat | 4, 6, 8 | Красный | Все раунды |
| Special | Нет веса | Синий | Только Раунды 2 и 3 |

Special карты НЕ раздаются в Раунде 1 (`currentRoundIndex < 1`).
Special карта не участвует в сравнении весов — только эффект.

### Правило дуэли

1. Атакующий → карта в AttackSlot (любая: Resource или Threat).
2. Защитник → карта в DefenseSlot (**ТОЛЬКО Resource карты**).
3. Вес защиты >= вес атаки → защитник победил.
4. Вес защиты < вес атаки → атакующий победил → получает пистолет.

**КРИТИЧЕСКОЕ ПРАВИЛО:** В DefenseSlot можно класть ТОЛЬКО Resource карты (серые, вес 2-8).
Threat карты (красные, вес 4/6/8) можно использовать ТОЛЬКО для атаки.
Если игрок пытается положить Threat карту в DefenseSlot → карта автоматически возвращается в руку.

Threat — это агрессивная карта. Отличие от Resource:
- Threat можно использовать только в атаке
- Если Threat атака не отбита → атакующий получает пистолет БЕЗ билета (более опасно)

### Результаты дуэли

| Результат | Условие | Пистолет | Билет |
|---|---|---|---|
| AttackerWins | Вес атаки > вес защиты | Атакующему | Атакующему |
| DefenderWinsBonus | Вес защиты > вес атаки | Никому | Защитнику |
| Draw | Веса равны | Никому | Никому |
| ThreatPassed | Threat атака не отбита | Атакующему | НЕТ |

При Draw и DefenderWinsBonus — роли меняются (защитник становится атакующим).

### Special карты

| Карта | Символ | Момент | Эффект |
|---|---|---|---|
| Cancel | C | До подсчёта | Отменяет Special карту противника. Вес не считается. |
| Steal | S | После победы | Победитель берёт случайную карту из руки проигравшего. |
| Duplicate | x2 | До подсчёта | Удваивает вес карты в слоте. После дуэли: сброс случайной карты из своей руки. |
| Exchange | <> | До подсчёта | Меняет карты в AttackSlot и DefenseSlot местами. Подсчёт после обмена. |
| Block | 0 | Текущий ход | Блокирует Special карты обоих игроков в этом ходу. |
| Prohibit | ! | Следующий мини-раунд | Запрещает Special карты в следующем мини-раунде. Снимается при StartMiniRound. |

Порядок применения: Block → Cancel → Exchange → Duplicate → подсчёт весов → Steal.

## 5. СТРЕЛЬБА

Владелец пистолета выбирает цель: враг или себя.

| Цель | Патрон | O2 | Результат |
|---|---|---|---|
| Враг | Live / Explosive | +30s | Урон врагу. +2 случайных билета стрелку. Ход СОХРАНЯЕТСЯ. |
| Враг | Blank | +10s | Нет урона. Ход переходит противнику. |
| Себя | Live / Explosive | +10s | Урон себе. Ход переходит противнику. |
| Себя | Blank | +10s | Нет урона. Ход СОХРАНЯЕТСЯ. Барабан прокручивается — позиция патронов меняется. |

Стратегия выстрела в себя Blank: если через Inspection известен следующий Live — можно его пропустить. Ход сохраняется → можно снова получить пистолет.

После каждого выстрела пистолет возвращается на GunSpot (kinematic Rigidbody).

## 6. ТАЙМЕР КИСЛОРОДА (O2)

Старт: 60s каждый раунд. O2 НЕ сбрасывается между мини-раундами. Накопленное время переходит.
Действия AI не пополняют O2 игрока. O2 сбрасывается ТОЛЬКО при переходе между Раундами.
При O2 = 0 → триггер Last Breath.

| Действие | +O2 | Тип |
|---|---|---|
| Взять карту с тумбы (раздача) | +5s | Пассивное |
| Поставить карту в AttackSlot | +10s | Активное |
| Поставить карту в DefenseSlot | +10s | Активное |
| Использовать Special карту | +10s | Активное |
| Использовать билет | +10s | Активное |
| Выстрел без урона | +10s | Активное |
| Выстрел с уроном | +30s | Рискованное |

## 7. БИЛЕТЫ

Билеты влияют ТОЛЬКО на HP, O2, пистолет, барабан.
На карты, веса и дуэль — НИКОГДА.

| Билет | Эффект | Свой ход | Нужен пистолет | Момент |
|---|---|---|---|---|
| Inspection | Тайно смотришь тип следующего патрона. | Нет | Нет | Любой |
| DoubleDamage | Следующий выстрел x2 урона. | Нет | Да | Любой |
| MedicalRation | +1 HP. Не выше максимума раунда (2/3/4). | Да | Нет | Свой ход |
| VoidTransaction | Отменяет результат текущей дуэли. Карты сброшены. Ход меняется. | Нет | Нет | Во время дуэли |
| Shuffle | Перемешивает барабан. Вся информация от Inspection теряется. | Нет | Нет | Любой |

Источники билетов:
- Раздача через тумбу
- AttackerWins → 1 билет атакующему
- DefenderWinsBonus → 1 билет защитнику
- Выстрел с уроном → +2 случайных билета стрелку

Ключевые комбинации:
- Inspection + DoubleDamage = гарантированные 2 HP урона
- Shuffle = контр на Inspection противника
- MedicalRation вовремя = предотвращает Last Breath
- VoidTransaction = отмена пистолета после проигрыша дуэли

## 8. LAST BREATH

Срабатывает автоматически. Максимум 1 раз за всю игру. Деактивируется после использования.

Триггер: HP = 0 ИЛИ O2 = 0.

Эффект:
- HP → 1 (игрок жив)
- Основной таймер O2 останавливается
- Запускается аварийный таймер: 15 секунд
- Состояние игры НЕ сбрасывается (дуэль, барабан, флаги — всё сохраняется)

За 15 секунд нужно:
- Опустошить барабан (все патроны выстрелены) ИЛИ
- Убить противника (HP врага = 0)

Успел → выживает. Основной таймер возобновляется с 0 (нужно сразу делать действия).
Не успел → Game Over. Финальный. Никаких шансов.

Второй триггер при 0 HP → немедленный Game Over без Last Breath.

## 9. УПРАВЛЕНИЕ

Игра должна быть полностью управляемой ТОЛЬКО мышью ИЛИ ТОЛЬКО клавиатурой.

| Действие | Мышь | Клавиатура |
|---|---|---|
| Выбор карты | Клик по карте | 1–5 (позиция в руке) |
| Размещение карты | Клик по слоту | Enter / Space |
| Отмена выбора | Scroll вниз | S / Escape |
| Взять пистолет | Клик по пистолету | Клавиша (TBD) |
| Переключить цель (себя/враг) | ПКМ | ПКМ |
| Использовать билет | Клик по билету | 1–5 |
| Камера → карты | Scroll вниз | S |
| Камера → слоты | Scroll вверх | W |
| Камера → назад | Scroll вниз / клик | S |

### Состояния камеры

| Состояние | Триггер | Описание |
|---|---|---|
| Free | Старт, S из PlayerSlots | Базовая позиция. Вид на стол и врага. |
| Cards | S или Scroll вниз из Free | Камера смотрит на карты в руке. |
| PlayerSlots | Клик по карте | Автопереход к зоне слотов игрока. |
| TableSlots | W из Free | Вид на стол, куда класть карты. |
| EnemySlots | W из TableSlots | Вид на сторону противника. |

Логика S/W:
- S: Free→Cards | PlayerSlots→Free | EnemySlots→PlayerSlots | Cards→нет реакции
- W: Cards→Free | Free→TableSlots | TableSlots→EnemySlots | EnemySlots→нет реакции

## 10. ЗАПРЕЩЁННЫЕ ПРАВИЛА (которых НЕТ в игре)

Если видишь в коде — это ошибка, нужно убрать:

- ❌ Threat карты в защите. НЕТ. Threat ТОЛЬКО для атаки. В DefenseSlot ТОЛЬКО Resource.
- ❌ AI накапливает патроны между мини-раундами. НЕТ. Патроны не накапливаются.
- ❌ O2 сбрасывается каждый мини-раунд. НЕТ. Только при смене Раунда.
- ❌ Атака на атаку (без защиты). НЕТ. Физически невозможно.
- ❌ Last Breath дважды. НЕТ. Только 1 раз.
- ❌ Special карты влияют на HP или O2. НЕТ. Только на дуэль.
- ❌ Пистолет при Draw или DefenderWinsBonus. НЕТ. Только AttackerWins и ThreatPassed.
- ❌ MedicalRation выше кап HP раунда. НЕТ. Кап = 2/3/4 по раунду.
